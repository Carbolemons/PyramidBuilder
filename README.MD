# PyramidBuilder

PyramidBuilder is a Java Console "app" that allows you to spawn neat little pyramids utilizing ANSI controls in the terminal.

## Running

You may use VSCode to launch and run using the default build task.

In Linux (Mostly any terminal with ANSI Support)

```
chmod +x ./build+run(linux).sh
./build+run(linux).sh
```

In Windows (Must be in a ANSI supported terminal. Windows 10 supports this.)

```
./build+run(windows).bat
```
